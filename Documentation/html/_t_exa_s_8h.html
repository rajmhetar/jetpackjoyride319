Detailed Description

Creates either an oscilloscope or a logic analyzer.

TExaS is an oscilloscope or a logic analyzer, sampling at 10 kHz

  • to create an 8-bit analog oscilloscope
    1. set adc12 to either ADC0 or ADC1
    2. set channel to 0 to 7
    3. scope ISR is 11 instructions, 1.5us, 1.5% overhead
  • to create a 7-bit digital logic analyzer
    1. set adc12 to 0, channel doesn't not matter
    2. create a function to read any 7-bit digital measurement, add bit7=1
    3. pass that function into TExaS_Init
Note
Works at 32MHz or 40MHz or 80 MHz

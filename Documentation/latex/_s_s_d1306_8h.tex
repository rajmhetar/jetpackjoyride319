\doxysubsection{Detailed Description}
Provide simple I/O functions for the SSD 1306 OLED. 

Use I2C to send an 8-\/bit code to the 1306 64x128 pixel OLED to display text, images, or other information. \begin{DoxyVersion}{Version}
RSLK2 
\end{DoxyVersion}

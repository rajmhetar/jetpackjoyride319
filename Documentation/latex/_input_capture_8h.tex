\doxysubsection{Detailed Description}
Software driver functions for input capture. 

PA12 is digital wave input, index 33 in IOMUX PINCM table.~\newline

\begin{DoxyItemize}
\item G0 in power domain PD0, so
\begin{DoxyItemize}
\item for 32MHz bus clock, SYSCLK clock is 32MHz
\item for 40MHz bus clock, SYSCLK clock is ULPCLK 20MHz
\item for 80MHz bus clock, SYSCLK clock is ULPCLK 40MHz~\newline

\end{DoxyItemize}
\end{DoxyItemize}

\doxysubsection{Detailed Description}
Creates either an oscilloscope or a logic analyzer. 

TExaS is an oscilloscope or a logic analyzer, sampling at 10 k\+Hz~\newline

\begin{DoxyItemize}
\item to create an 8-\/bit analog oscilloscope
\begin{DoxyEnumerate}
\item set adc12 to either ADC0 or ADC1
\item set channel to 0 to 7
\item scope ISR is 11 instructions, 1.\+5us, 1.\+5\% overhead
\end{DoxyEnumerate}
\item to create a 7-\/bit digital logic analyzer
\begin{DoxyEnumerate}
\item set adc12 to 0, channel doesn\textquotesingle{}t not matter
\item create a function to read any 7-\/bit digital measurement, add bit7=1
\item pass that function into TExa\+S\+\_\+\+Init
\end{DoxyEnumerate}
\end{DoxyItemize}

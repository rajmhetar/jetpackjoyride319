This function assumes that all of that has been stripped out, and the array image\mbox{[}\mbox{]} has one 16-\/bit halfword for each pixel to be displayed on the screen (encoded in reverse order, which is standard for bitmap files). An array can be created in this format from a 24-\/bit-\/per-\/pixel .bmp file using the associated converter program. (x,y) is the screen location of the lower left corner of BMP image~\newline
